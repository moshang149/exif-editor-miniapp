### 关键设计决策

1. **EXIF 处理策略**: 客户端 Canvas 为主，服务端注入为降级——保护用户隐私，不依赖网络
2. **模板存储**: 纯本地 `wx.Storage`，离线可用，6 个内置模板（覆盖 iPhone/Canon/Leica/Sony/Fujifilm/Nikon）
3. **撤销/重做**: 命令模式，内存栈最多 20 步
4. **架构分层**: 视图层 → 组件层 → 服务层 → 工具层 → 微信 API，严格解耦
5. **预算控制**: 主包 ≤ 1.5MB，为后续迭代预留空间
